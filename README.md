# Tienda Virtual tipo Shein

Proyecto desarrollado con fines académicos durante la formación como
Tecnóloga en Análisis y Desarrollo de Software – SENA.

## Descripción
Aplicación web tipo e-commerce inspirada en plataformas como Shein.
Incluye catálogo de productos, carrito de compras y estructura de vistas
para la simulación de una tienda virtual.

## Tecnologías utilizadas
- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (entorno local)

## Funcionalidades
- Visualización de productos
- Carrito de compras
- Estructura MVC
- Vistas dinámicas

## Ejecución
Este proyecto fue desarrollado y ejecutado en entorno local utilizando XAMPP.
Para su funcionamiento se requiere servidor local y configuración de base de datos.
